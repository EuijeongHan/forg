# forG 고지문 모음 (봇·화면 삽입용)

> 용도별 짧은 고지문. 약관·처리방침의 핵심을 사용자 접점에 노출하기 위한 문구.
> 코드 반영은 별도 PR로(아래 "반영 위치" 참조).

---

## 1. `/start` 환영 메시지 하단 (봇 최초 접점 — 필수)

```
ℹ️ forG는 DART 공시 정보를 요약해 전달하는 참고용 도구입니다.
투자 자문·종목 추천 서비스가 아니며, AI 요약에는 오류·지연이 있을 수 있습니다.
투자 판단 전 반드시 DART 원문을 확인하세요.
이용약관·개인정보처리방침: <링크>
```

## 2. 알림 메시지 말미 (현행 유지 + 보강안)

현행(summarizer.SYSTEM_PROMPT가 부착):
```
⚠️ 본 요약은 AI 참고용입니다. 투자 판단 전 원문을 확인하세요.
```
보강안(정형 카드 등 AI 문구가 안 붙는 경로 포함, notifier 레벨에서 일괄 부착 검토):
```
⚠️ AI 요약·자동 분류에는 오류가 있을 수 있습니다. 원문 확인 필수 · 투자 조언 아님
```

## 3. 설문·모집 안내용 (Google Forms 도입부와 별도 배포 글)

```
forG는 개인이 운영하는 무료 베타 서비스로, 공시 확인 업무를 돕는 참고 도구입니다.
투자자문 서비스가 아니며, 서비스 안내 외 목적으로 연락처를 사용하지 않습니다.
```

## 4. 향후 웹 화면 푸터용 (Stage 8 이후)

```
forG는 금융감독원 전자공시시스템(DART)의 공시 정보를 원천으로 하는 정보 제공
서비스입니다. 본 서비스의 요약·분류는 참고용이며 투자권유·자문이 아닙니다.
공식 정보는 dart.fss.or.kr 원문을 확인하십시오.
```
